Describe how search engines work

Search engines work by finding hundreds of pages using their own web techniques. A search engine navigates the web by downloading web pages and following links.This link then follows to new pages to discover content made available. Search engines are searchable databases of web content. Search engines are made up of two  parts:  search index and search algorithms. Search index is a library of information about webpages. Lastly search algorithms are computer programs that are tasked with matching results from the search index.

What constitutes a good heading hierarchy and include one example with more than one heading tag

HTML header tags follow a hierarchy, from 1 to 6. H1 tags are used to organize the most important text, such as the main theme or title. H2 and H3 tags are commonly used as subheadings. Finally, H4, H5, and H6 tags may be used to provide structure within those subsections of the page. Header tags are good for users and search engines. For your users, they give them a structral preview of the content they’re about to read.For search engines , they provide context on what your page is about.
