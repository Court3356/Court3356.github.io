Explain ways to implement preprocessing

Client-side

Client side refers to the processes that are carreid out on the users device, typically in the users web browser.These processes are executed after the web app has been delivered to the users device.They can include tasks such as rendering and displaying a web page

Server-side

Server side processing optimies web preformance by processing and filtering data before sending it out.This processes is carried out by a web server.These processes are typically executed by the server before the website or web app is delivered to the user's device.

Define and describe the mechanisms used in LESS

When using preprocessing compliers such as LESS,web developers gain new features to create well designed websites.Stylesheets are constantly growing and becoming more complex.Less allows users to have more freedom by using variables,mix-ins,functions,nested variables and operators. These preprossesing compliers bring new tools to your site.

LESS

LESS stands for Linear Style Sheets. It is a backward compatible language extension for CSS.When installed the user can create CSS using fewer lines of code by adding in variables.LESS is a source that is open to work with all versions of CSS.

Variables

When using LESS variables they are designed through the use of the @,the value is then placed in a colon ":".When using this variable it reduces the number of times a value is used

Nesting

Nested rules are a set of properties of the CSS that allow the properties of one class to be used for another class, following with the same name as the property.When using LESS you can class or id and store several values using style.less and compose that into the style.css file

Operators

LESS gives arithmetic operations like subtraction,multiplication,addition and division.When the property is used it makes doing math equations easer. This can be applied to anything that uses numbers such as variable,numbers and colors.

Function

LESS provides a variety of functions which transform colors,manipulate strings and do math.Using them is pretty simple.
